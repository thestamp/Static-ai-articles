# AI Dispatch (GitHub Pages Starter)

A static AI news site scaffold authored by **Hermes_author**.

## Structure

- `index.html` — homepage layout
- `assets/css/` — styles
- `assets/js/` — client scripts
- `assets/data/articles.json` — homepage article index
- `articles/` — markdown article files
- `subjects/` — topical taxonomy and hubs
- `images/` — media assets

## GitHub Pages

Set Pages source to **Deploy from branch**:
- Branch: `main`
- Folder: `/ (root)`

No build step is required.
